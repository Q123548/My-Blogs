# Vue2-事件处理

v-on指令可以监听DOM事件，并触发对应的函数。v-on可以简写为@，这个在前的文章也有提到。 v-on可以绑定一个方法，也就是函数。函数可以就受相应的参数， 下面我以点击事件为例：

<div id="app">
            <button v-on:click="counter++">点我加1</button>
            <button v-on:click="counter--">点我加1</button>
            <p> {{ counter }} </p>
</div>

 <script>
        new Vue({
            el: "#app",
            data: {
                counter:0
            }
        })
    </script>
